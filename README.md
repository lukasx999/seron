# Seron Programming Language

## Features

- Compiled
- Static Typechecking
- Strong Typing
- **No** dependencies

Currently, the only supported backend is x86_64 NASM.

## Build Requirements

- C11 Toolchain
- Nothing else!

## Building Compiler

```
$ git clone https://github.com/lukasx999/seron.git
$ cd seronc/src
$ make
```
